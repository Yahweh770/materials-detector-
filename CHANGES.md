# Изменения в приложении "Учёт материалов"

## Добавленные функции (ранее отсутствовали)

### 1. `show_tree_context_menu(event)`
- Показывает контекстное меню при клике правой кнопкой мыши по таблице
- Автоматически выбирает элемент под курсором
- Сохраняет координаты события для последующего использования

### 2. `copy_row_to_clipboard()`
- Копирует всю строку таблицы в буфер обмена
- Формирует данные с заголовками колонок
- Разделитель - табуляция (удобно для вставки в Excel)
- Показывает уведомление об успешном копировании

### 3. `copy_cell_to_clipboard(event=None)`
- Копирует содержимое конкретной ячейки в буфер обмена
- Определяет ячейку по координатам клика
- Если не удалось определить конкретную ячейку, копирует первое значение
- Показывает уведомление с скопированным значением

### 4. `search_data()`
- Поиск по всем полям записей
- Регистронезависимый поиск
- Отображает только найденные совпадения
- Показывает количество найденных записей
- Автоматический поиск при вводе текста (trace на search_var)

### 5. `clear_search()`
- Очищает поле поиска
- Восстанавливает отображение всех данных
- Обновляет информацию о просроченных документах

## Улучшения кода

### 1. Документация
- Добавлен подробный docstring для модуля
- Добавлены docstring для класса и всех методов
- Комментарии на русском языке для лучшей читаемости

### 2. Структура кода
- Добавлена переменная `_context_menu_event` для хранения события контекстного меню
- Улучшена обработка ошибок в методе `copy_cell_to_clipboard`
- Добавлена защита от исключений при определении индекса колонки

### 3. Пользовательский интерфейс
- Улучшены сообщения об успешных операциях
- Добавлены информационные сообщения о количестве найденных записей
- Контекстное меню теперь правильно работает с координатами клика

## Исправленные проблемы

1. **Отсутствие реализации методов** - все методы, которые вызывались в интерфейсе, но не были реализованы, теперь работают полноценно

2. **Поиск не работал** - метод `search_data()` отсутствовал, теперь реализован с автоматическим поиском при вводе

3. **Контекстное меню не функционировало** - добавлены все необходимые обработчики для работы контекстного меню

4. **Копирование данных** - реализовано копирование как отдельных ячеек, так и целых строк

## Как использовать новые функции

### Поиск
1. Введите текст в поле поиска на панели инструментов
2. Результаты фильтруются автоматически
3. Нажмите "🔍 Очистить" для сброса фильтра

### Копирование через контекстное меню
1. Кликните правой кнопкой мыши по строке таблицы
2. Выберите "Копировать строку" или "Копировать ячейку"
3. Данные будут скопированы в буфер обмена

### Горячие клавиши
- `Ctrl+C` - копирование выделенной строки или содержимого текстового поля
- `Ctrl+V` - вставка в текстовое поле
- `Ctrl+X` - вырезание в текстовом поле
- `Delete` - удаление выделенной строки
